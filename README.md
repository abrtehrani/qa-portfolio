# QA Testing Report – SauceDemo Application

This project documents manual QA testing of the SauceDemo web application.

## Project Overview
The purpose of this project is to demonstrate real-world manual QA skills including test design, execution, defect reporting, and exploratory testing.

## Application Under Test
- SauceDemo: https://www.saucedemo.com/

## Testing Scope
- Login functionality
- Product listing
- Shopping cart
- Checkout process

## Testing Types
- Functional Testing
- Exploratory Testing
- API Testing (basic via Postman)
- UI & Validation Testing

## Deliverables
- Test Strategy
- Test Cases
- Bug Reports
- Exploratory Testing Sessions
- Test Execution Summary

## Tools Used
- Postman
- GitHub
- Markdown / Excel (for test cases)

## Summary
This project demonstrates end-to-end manual QA activities including test design, execution, defect reporting, and basic API validation.
